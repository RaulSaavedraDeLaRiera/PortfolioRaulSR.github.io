Monkey Delivery

Videojuego 2D con foco en UI/UX y arte

2022 - 2023 Desarrollo 2D

Descripción del Proyecto

Monkey Delivery es un videojuego 2D desarrollado en C++ utilizando la librería SDL. El proyecto se centra en el apartado artístico, la interfaz de usuario y la experiencia del usuario, creando una experiencia visual atractiva y jugable.

El juego presenta mecánicas de entrega y navegación en un mundo 2D con un diseño visual cuidado y una interfaz de usuario intuitiva.

Tecnologías Utilizadas

C++

Lenguaje de programación principal

SDL

Librería multimedia

SDL_Image

Manejo de imágenes

SDL_Mixer

Sistema de audio

SDL_TTF

Renderizado de texto

Características Principales

  • Motor de juego 2D personalizado
  • Sistema de sprites y animaciones
  • Interfaz de usuario intuitiva
  • Sistema de física básico
  • Música y efectos de sonido
  • Sistema de puntuación
  • Múltiples niveles
