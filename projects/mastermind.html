Mastermind

Aplicación Android con funciones nativas y sensores

2022 - 2023 Desarrollo Android

Descripción del Proyecto

Mastermind es una aplicación Android desarrollada en Android Studio que implementa el clásico juego de lógica con características modernas. La aplicación hace uso de funciones nativas del dispositivo y sensores para crear una experiencia interactiva única.

El juego incluye un sistema de anuncios integrado y aprovecha las capacidades del dispositivo móvil para ofrecer una experiencia de usuario rica y envolvente.

Características Principales

  • Implementación del juego Mastermind clásico
  • Uso de sensores del dispositivo (acelerómetro, giroscopio)
  • Interfaz de usuario nativa de Android
  • Sistema de puntuación y estadísticas
  • Integración de anuncios AdMob
  • Guardado local de progreso
  • Diferentes niveles de dificultad
